# 图片粘贴裁剪接续工具

一个基于 HTML5 Canvas 的图片处理工具，支持粘贴图片、拖拽选择区间删除接续、矩形马赛克打码、箭头标注、数字标注等功能。

## 功能特性

- **图片粘贴** - 按 `Ctrl + V` 直接粘贴剪贴板中的图片
- **区间裁剪接续** - 拖拽选择水平区间，删除后自动接续
- **多种拖拽模式**：
  - 普通拖拽 = 重置选中区间
  - `Ctrl` + 拖拽 = 增加选中区间
  - `Shift` + 拖拽 = 从已有区间减掉
- **矩形马赛克** - 点击"矩形马赛克"按钮后，在图片上拖拽矩形区域进行打码（快捷键 `Ctrl + M`）
  - 采用组合式马赛克算法，使用块平均色，效果更自然
- **画箭头** - 点击"画箭头"按钮后，在图片上拖拽绘制红色箭头（快捷键 `A`）
- **数字标注** - 点击"数字标注"按钮后，点击图片放置递增数字圆圈（快捷键 `N`）
  - 数字自动递增（1, 2, 3...），方便连续标注多个位置
- **缩放功能** - 按住 `Ctrl` + 鼠标滚轮可缩放查看，点击"缩放还原 1:1"恢复原尺寸
- **撤销/重做** - 支持 `Ctrl + Z` 撤销，`Ctrl + Y` 重做
- **下载结果** - 处理后的图片可保存为 PNG 格式

## 使用方法

1. 打开页面
2. 复制任意图片到剪贴板
3. 在页面中按 `Ctrl + V` 粘贴图片
4. 拖拽鼠标选择要删除的水平区间
5. 点击"删除并接续"按钮执行操作
6. 如需打码，点击"矩形马赛克"后在图片上拖拽选择区域
7. 如需标注，点击"画箭头"或"数字标注"后进行操作
8. 点击"下载结果 PNG"保存处理后的图片

## 技术实现

- 纯前端实现，无需后端服务
- 使用 HTML5 Canvas 进行图像处理
- 支持历史记录管理（最多 30 步）
