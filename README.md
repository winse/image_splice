# ImageSplice Pro | 图片粘贴裁剪接续工具

一个基于 HTML5 Canvas 的图片处理工具，支持粘贴图片、拖拽选择区间删除接续、矩形马赛克打码、箭头标注、数字标注等功能。

## 功能特性

- **工具模式切换** - 支持选择删除区间、马赛克、画箭头、数字标注四种工具模式，点击按钮或按快捷键 `S`/`M`/`A`/`N` 快速切换
- **区间裁剪接续** - 使用"选择删除区间"工具（`S`）拖拽选择水平区间，按 `Delete` 键删除后自动接续
- **多种拖拽模式**：
  - 普通拖拽 = 重置选中区间
  - `Ctrl` + 拖拽 = 增加选中区间
  - `Shift` + 拖拽 = 从已有区间减掉
- **矩形马赛克** - 切换到"矩形马赛克"工具（`M`），在图片上拖拽矩形区域进行打码
  - 采用组合式马赛克算法，使用块平均色，效果更自然
- **画箭头** - 切换到"画箭头"工具（`A`），在图片上拖拽绘制渐进式红色箭头
- **数字标注** - 切换到"数字标注"工具（`N`），点击图片放置递增数字圆圈
  - 数字自动递增（1, 2, 3...），方便连续标注多个位置
- **标注元素可拖拽** - 在箭头或数字标注模式下，可直接拖拽移动已添加的标注元素，选中后按 `Delete` 删除
- **撤销/重做** - 支持 `Ctrl + Z` 撤销，`Ctrl + Y` 重做
- **下载结果** - 处理后的图片可保存为 PNG 格式

## 使用方法

1. 打开页面，复制任意图片到剪贴板，按 `Ctrl + V` 粘贴图片
2. 使用"选择删除区间"工具（快捷键 `S`）拖拽选择要删除的水平区间
3. 按 `Delete` 键执行删除并接续
4. 切换到"矩形马赛克"工具（`M`）进行打码
5. 切换到"画箭头"（`A`）或"数字标注"（`N`）工具进行标注，支持拖拽移动标注
6. 点击"下载结果 PNG"保存处理后的图片

## 技术实现

- 纯前端实现，无需后端服务
- 使用 HTML5 Canvas 进行图像处理
- 支持独立标注层，标注元素可独立管理、撤销/重做、拖拽移动
- 支持历史记录管理（最多 30 步）

## 快捷键参考

| 快捷键 | 功能 |

| `Ctrl + V` | 粘贴图片 |
| `S` | 切换到"选择删除区间"工具 |
| `M` | 切换到"矩形马赛克"工具 |
| `A` | 切换到"画箭头"工具 |
| `N` | 切换到"数字标注"工具 |
| `Delete` | 删除选中的区间或标注元素 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Ctrl + 滚轮` | 缩放画布 |
